Chow.

November 2020

Chow. is a project I made for my human-computer interaction course. One of the labs for that course was to put together a prototype of a food delivery app using Figma. It was primarily focused on the interactive aspects of creating a prototype, as well as applying the HCI concepts we learned in class to create a easy and comfortable user experience. However, I had a lot of fun with the visual design as well.

Through this project, I learned a few things about Figma that I didn't know before hand, mostly about how to create more interactive prototypes. For example, implementing both horizontal and vertical scrolling elements within a frame, swapping elements vs. using overlays, and more.

Nielsen's "Who-ri-sticks"?

For this project we also had to point out some examples of Nielsen's Heuristics within our app prototypes to prove that we, ya know, actually learned something. Certain things such as consistency, affordances, flexibility of navigation options, visibility of elements, error prevention, etc. were all considerations that I made while designing the app and prototype.
